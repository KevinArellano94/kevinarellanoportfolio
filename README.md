## Available Scripts

In the project you will use the following scripts:

### `cd { folder location }`

### `ls`

### `npm -v`

### `npx -v`

### `yarn -v`

### `yarn create react-app { projectname }`

*Note: there is no longer a dash "-" in between "create" and "react".

### `cd { inside project folder }`

### `yarn add router-react-dom`

### `yarn start`

### `yarn build`
